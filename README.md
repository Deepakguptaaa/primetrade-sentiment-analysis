# 📊 PrimeTrade AI – Sentiment vs Trading Performance Analysis

## 🚀 Project Overview
This project analyzes the relationship between Bitcoin market sentiment (Fear/Greed Index) and trader performance using historical Hyperliquid trading data.

The goal is to identify whether market sentiment influences:
- Trader profitability
- Risk behavior (leverage & position size)
- Trading direction (long/short bias)

---

## 📂 Dataset
1. Fear & Greed Index (Bitcoin sentiment)
2. Hyperliquid Historical Trading Data

---

## 🧠 Key Insights
- Trader profitability varies across sentiment regimes
- Greed phases show higher risk-taking behavior
- Fear phases show more conservative trading
- Sentiment has measurable impact on trading performance
