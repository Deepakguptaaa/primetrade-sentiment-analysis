# PrimeTrade AI Sentiment Analysis

## Objective
Analyze relationship between Bitcoin market sentiment and trader performance.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Profitability changes with Fear/Greed sentiment
- Risk-taking increases during Greed phase
- Market sentiment impacts trading behavior

## Dataset
- Fear & Greed Index
- Hyperliquid trading data
